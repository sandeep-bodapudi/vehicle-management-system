# Vehicle Management System

A full-stack fleet management dashboard built with the MERN stack.

## Features
- Track vehicles, assign drivers, and manage service/maintenance logs
- Admin dashboard with role-based access
- JWT-based authentication
- MongoDB schemas designed to handle relationships between vehicles, drivers, and service records
- RESTful API built with Node.js and Express.js
- React frontend with real-time state management

## Tech Stack
**Frontend:** React.js  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Auth:** JWT  
**Tools:** Postman, Git
